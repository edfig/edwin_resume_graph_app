# Edwin Resume Graph App

Streamlit app visualizing a career knowledge graph.

Quick start:

1. Create a virtual environment and install requirements:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. Run locally:

```bash
streamlit run app.py
```

To deploy on Streamlit Community Cloud, push this repository to GitHub and connect the repo in the Streamlit deploy UI.
